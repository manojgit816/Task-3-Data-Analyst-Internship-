
# Task 3 – SQL Basics (Filtering, Sorting & Aggregations)

## Files Included
- **queries_task3.sql**: Contains all SQL queries used for analysis.
- **sales_summary.csv**: Exported summary report (sales by category).
- **README.md**: Explanation of queries and outputs.

## Query Explanation
1. **SELECT \*** – View all rows to understand data.
2. **COUNT(*)** – Verify total number of records.
3. **WHERE** – Filter records (Technology category).
4. **ORDER BY** – Sort data to find top-selling products.
5. **GROUP BY** – Aggregate sales and profit by category.
6. **HAVING** – Filter aggregated results (Sales > 100000).
7. **Regional Summary** – Sales grouped by region.
8. **BETWEEN** – Date range filtering for reports.
9. **LIKE** – Pattern search on customer names.

## Outcome
This task builds confidence in writing real-world SQL queries using filtering,
sorting, aggregation, and grouping techniques.
